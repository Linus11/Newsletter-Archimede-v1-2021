Le BAHR : de Flora vers Heurist

   Á la suite de sa migration vers une nouvelle solution logicielle, Heurist, la base de données Flora sera fermée en écriture comme en lecture. L'ensemble des ressources de Flora seront injectées alors dans la nouvelle base, hébergée sur la plateforme Huma-Num, la nouvelle base de données assurera une meilleure pérennisation des ressources numériques du BAHR.

Nakala refait peau neuve

   L'interface de dépôt et de gestion de données Nakala a été entièrement revue, elle propose un accès simple, une nouvelle façon plus intuitive d'organiser vos données et une meilleure visibilité des enregistrements via un tableau de bord ergonomique. Ainsi, le pack "Nakalona" devient "Nakala-Press ».

   Nakala permet également de gérer les identifiants pérennes de type DIP, en générant un DOI (l’équivalent d’un Handle ou Ark) pour chaque ressource. La diffusion se fait via le "triple store" et les formats OAI-PMH permettent une large diffusion sur les entrepôts de stockage et d'archivage (Hals, BNF, etc.).

Avec Omeka, diffusez vos données en un clic !

Le format Omeka vous offre la possibilité de redonner vie à vos jeux de données pour plus de visibilité et une meilleure représentation (site web). En voici les avantages :

  •   Obtenir une version documentée de la base de données en format Omeka
  •  Chaque dataset est présenté individuellement de façon inhérente à la base de données
  •  En définitive, une simple importation des fichiers Omeka qui vient enrichir votre bibliothèque numérique géo-référencée et autres médias (images, vidéos, schémas, etc.)

Retour sur les nouveautés de 2020 !

   Depuis sa version 4.2 (en attendant le financement de la version 5), ArkeoGIS continue d’asseoir sa maturité et se dote de nouvelles fonctionnalités afin d'améliorer « l’expérience utilisateur » et la capacité des utilisateurs à interagir avec l’outil, aussi bien sur le site web que sur l’application.

   Souvent comparé à un agrégateur, ArkéoGIS est bien plus que cela. Il s’agit d’un vrai logiciel puissant et robuste, doté d'un système lui permettant de croiser les données, les moissonner et les rendre réutilisables dans un format standardisé.
